1) Download XAMPP Apache Control for PHP and sql part

2) Extract the file and copy "exam" folder

3) Paste inside root directory

4) Open XAMPP control Panel and make Sure that XAMPP PHP and SQL have been started 

5) Open PHPMyAdmin (http://localhost/phpmyadmin)

6) Create a database with the name 'register'

7) Import register.sql file(given inside the zip package folder)

8) Run the script http://localhost/exam
