About

This demo website project is developed by Debre Birhan University Third year COMPUTER SCIENCE Group 3 & 8 students.

ONLINE EXAMINATION SYSTEM

It is a page where Students can enroll themselves to an exam that they want to take and test their capabilities based on different exam subjects.

In this project we tried to demonstrate our web by using PHP,MYSQL,HTML,CSS.
Using MYSQL it will record the registration data of a user in a database called 'register' and a table called 'examdb'.

But this web is only student side server, there should be another teacher side server where teachers can load different exams , block and unblock students , set timer of an exam and check student result of an exam etc..
